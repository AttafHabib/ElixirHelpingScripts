# ElixirHelpingScripts

Clone the Repo
```
  git clone https://github.com/AttafHabib/ElixirHelpingScripts.git
```

In the cloned repo, run IEx shell
<br />
Run
<br />
    ```
    c ["Helpers.ex"]
    ```

    ```
    Helpers.getAssigns(path_to_file)
    Helpers.get_comment_lines(path_to_file)
    
    ```
